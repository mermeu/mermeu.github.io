9/10/2018 + 11/10/2018 Wat is onderzoek bij CMD? Onderzoek is een belangrijk deel van het ontwerpproces en is in essentie slechts het vinden van een antwoord. Omdat het ontwerpproces iteratief is, doe je dus telkens opnieuw onderzoek. Onderzoeken en ontwerpen vormen dan ook een symbiotische relatie, ze zijn van elkaar afhankelijk. Bij CMD is het ten eerste belangrijk dat ik leer hoe ik bestaand onderzoek uit verschillende vakgebieden kan vinden en op waarde kan schatten (en ga begrijpen hoe een ontwerp-onderzoek verschilt van andere vormen van onderzoek).
